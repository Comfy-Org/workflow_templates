# Qwen-Image

Qwen-Image (also known as Qwen-VL) is a vision-language model series from Alibaba's Qwen team, capable of understanding and editing images.

## Model Variants

### Qwen2-VL

- Multimodal vision-language model
- Image understanding and description
- Available in 2B, 7B, and 72B parameter sizes

### Qwen-Image-Edit

- Specialized for image editing tasks
- Instruction-following for precise edits
- Supports inpainting, outpainting, and style transfer

## Key Features

- Strong visual understanding
- Multi-language support
- Instruction-based editing
- High-quality image generation and manipulation

## Hardware Requirements

- 2B model: 6GB VRAM
- 7B model: 16GB VRAM
- 72B model: Multi-GPU setup required

## Common Use Cases

- Image editing based on text instructions
- Visual question answering
- Image-to-image transformation
- Content-aware modifications
- Style transfer and artistic effects
